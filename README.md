# NASA Media
React app to view NASA images and a "Picture of the Day".

# Technologies Used
> React<br>
> APIs<br>
> Flexbot<br>
> SASS

# Getting Started
Fork and clone this repository into your computer. Open in your favorite code editor. Type <strong>npm start</strong> into your terminal to run the server. If not cloning, click this link to go to the site:

Click the "See Into the Stars!" button in the center of the landing page to be redirected to the NASA Photo of the Day. Use the navigation bar to view other pages of the site. The "Search Gallery" link takes you to a search bar that can be used to find pictures from the NASA image database using keywords. The "About" link shows information about the site.
